Features

Everything you need for fast, reliable file synchronization.

⚡

BLAKE3 Hashing

Cryptographic SIMD-accelerated hashing at 10 GB/s per core. Multi-threaded across all CPU cores.

🧩

Content-Defined Chunking

Gear-hash rolling function splits files at content boundaries. Only changed chunks are transferred.

🪶

Zero-Copy mmap

Memory-mapped file I/O eliminates redundant copies. Files go directly from disk to hash to network.

🔒

TLS 1.3 Network

TCP client-server architecture with optional TLS encryption via tokio-rustls.

📊

Delta Sync

96%+ bandwidth savings. A 1-byte change in 800 KB transfers only 8–33 KB.

🌍

Lower Carbon Footprint

47% less energy per sync. At cloud scale, saves 15+ tons CO₂/year.

Get Started in 30 Seconds

One command to build, then sync like rsync — but faster.

# Clone and build
$ git clone https://github.com/vedprakashpandey/resync-rs.git
$ cd resync-rs && cargo build --release

# Sync a directory (archive mode, 8 threads)
$ ./target/release/resync -a -j 8 /src/ /dst/

# Network sync
$ resync serve --listen 0.0.0.0:8730 --root /backup/
$ resync push /data/ --server 10.0.0.1:8730
